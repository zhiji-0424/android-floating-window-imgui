## 项目简介

- 在 Android 设备上运行并显示 ImGui 窗口。

## 获取和安装

1. 克隆本项目，并使用 Android Studio 或其他兼容软件打开。
2. 编译并安装到 Android 设备，授予悬浮窗权限后即可使用。
